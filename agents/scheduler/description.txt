This agent helps university students build their academic schedule by computing a list of courses they are eligible to enroll in. It uses a custom algorithm to analyze the student's completed coursework, cross-references this against degree requirements and prerequisites, and filters by courses offered in the current academic quarter. Once the eligible course set is computed, the agent can optionally refine suggestions based on user preferences, such as topic focus or scheduling constraints. It can also confirm when courses have been successfully added to a draft schedule.
